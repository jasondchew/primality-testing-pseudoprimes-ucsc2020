# Primality Testing with Pseudoprimes
**Senior Capstone Project | University of California, Santa Cruz**

This project explores the mathematical foundation and computational implementation of primality testing using Pseudoprime numbers. By leveraging Fermat's Little Theorem, we demonstrate how to efficiently verify the primality of 100+ digit numbers—a critical process for secure data encryption in RSA Cryptography.

## Key Features
* **Mathematical Analysis:** In-depth exploration of Fermat Pseudoprimes and Carmichael numbers.
* **Custom Python Implementation:** Optimized functions for modular exponentiation and primality verification.
* **Data Visualization:** Comparative analysis of testing efficiency using `matplotlib`.
* **Interactive Learning:** A comprehensive Jupyter Notebook designed for both technical review and educational demonstration.

## Links
* `primality_testing_pseudoprimes_ucsc2020.ipynb`: The primary research and code environment.
* `images/`: Visualizations of prime distribution and algorithm performance.

## Tech Stack
* **Language:** Python 3.x
* **Libraries:** Matplotlib
* **Environment:** Jupyter Notebook

## Key Insights
* Traditional trial division is nearly impossible for RSA-level primes
* Pseudoprime testing provides a probabilistic approach that handles massive integers in milliseconds
